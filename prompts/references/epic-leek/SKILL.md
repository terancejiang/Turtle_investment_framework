# 史诗级韭菜投资方法论参考框架

基于B站UP主"史诗级韭菜"（粉丝33.6万，投资实盘1100+日记录）的投资体系提炼。其自我定位为"格施风格古典捡垃圾"——格雷厄姆(Graham)+施洛斯(Schloss)风格的古典价值投资，做估值修复而非费雪式成长股。

**核心哲学**：不做导数交易，只做定性判断。市场认为你是负的，我压住你是正的——赚定性级别的估值修复，而非高估值高成长的导数逻辑。

## When to Use

- 用户在分析个股时需要参考"估值修复"或"逆向投资"框架
- 用户讨论"双负溢价"选股策略
- 用户需要计算"可支配现金流"（区别于传统自由现金流）
- 用户需要判断净现金是否可作为"资产积累"（安全垫）
- 用户需要处理少数股东权益对现金流的影响
- 用户讨论特许经营/垄断型商业模式的估值溢价
- 用户分析烟蒂股的事件驱动机会
- 用户讨论系统性风险下的防守策略
- 用户提及"史诗级韭菜"或其投资观点
- 用户使用 /epic-leek-methodology 命令

## 核心方法论体系

本框架包含两大并行体系和多个横向分析工具：

### 体系一：龟龟投资法（质量型价值投资）

偏向有合理商业模式的低估公司，核心因子：
1. **商业模式评估**（决定仓位上限）
2. **可支配现金结余**（比传统FCF更保守）
3. **穿透回报率 GG%**（综合回报率指标）
4. **资产积累与安全边际**

### 体系二：烟蒂策略（静态价值型投资）

纯粹格雷厄姆/施洛斯式，核心因子：
1. **存量资产垫**（净资产对股价的支撑）
2. **正现金流入**（避免价值陷阱）
3. **事件驱动**（私有化、要约收购等）
