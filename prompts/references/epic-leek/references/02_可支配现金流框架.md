# 可支配现金流框架

## 核心公式

UP主自创的现金流指标，比传统自由现金流（FCF）更保守：

```
可支配现金流 = (经营现金流 - 全部资本开支 - 租赁本金偿还) × 归属母公司股权比例
```

### 与传统FCF的关键区别

| 项目 | 传统FCF | UP主的可支配现金流 |
|------|---------|-------------------|
| 资本开支 | 区分维护性/扩张性 | **不区分，全扣** |
| 租赁支付 | 可能忽略 | **扣除租赁本金偿还** |
| 少数股东 | 通常忽略 | **乘以实际持股比例** |
| 保守程度 | 中等 | **更保守** |

> "我不区分维护性资本开支和扩张性资本开支。你说今年这个设备是维护的不是扩张的？我不信。反正都是花出去的钱，全部扣掉。"

## 少数股东权益调整（最关键的差异）

**问题场景：** 合并报表将子公司100%并表，但母公司可能只持有65%。报表上的经营现金流是100%的，但实际上只有65%属于母公司股东。

**案例：中国食品（00506.HK）**
- 中粮集团持有中国食品约65%
- 报表OCF = 100%合并数
- 实际可支配 = 报表OCF × 0.65

**何时需要调整：**
- 少数股东权益占比 > 10% 时必须调整
- 持股比例在年报"公司架构"或"重大子公司"章节查找
- 如有多层嵌套，需穿透计算实际持股比例

## 四步计算流程

### Step 1: 提取经营现金流（OCF）
- 来源：现金流量表 → 经营活动产生的现金流量净额
- 注意：检查是否有大额一次性经营现金流入（如预收款大增）

### Step 2: 扣除全部资本开支
- 来源：现金流量表 → 投资活动 → 购建固定资产、无形资产和其他长期资产支付的现金
- **不做任何维护性/扩张性区分，全额扣除**

### Step 3: 扣除租赁本金偿还
- 来源：现金流量表 → 筹资活动 → 偿还租赁负债本金
- IFRS 16下租赁拆分为本金（筹资活动）和利息（经营/筹资活动）
- 只扣本金部分

### Step 4: 乘以归属母公司持股比例
- 来源：年报 → 公司架构 / 重大子公司持股比例
- 如果母公司100%持股所有重大子公司 → 比例 = 1.0，可跳过此步

## 穿透回报率计算

基于可支配现金流推算股东回报：

```
穿透回报率 = (注销型回购金额 + 预期分红) / 市值（或成本市值）
```

### 阈值标准

| 比较基准 | 最低要求 | UP主个人标准 |
|----------|----------|-------------|
| 无风险利率(Rf) | ≥ 2 × Rf | ≥ 4 × Rf |
| 当前Rf ≈ 2% | ≥ 4% | ≥ 8% |

### 注意事项
- **只计注销型回购**：库存股回购不算（可能用于员工激励，不减少总股本）
- **预期分红基于可支配现金流**：不是基于会计利润
- **分母选择**：用市值而非企业价值（EV），因为回购/分红的受益者是股东，对应股权价值

> "穿透回报率至少要两倍无风险，这是底线。我个人要求四倍以上才考虑。两倍只是说这个东西不算贵，四倍才是我觉得便宜。"

## 可支配现金流的局限性

1. **不适用于高增长期**：大量扩张性资本开支会严重压低可支配现金流
2. **周期底部扭曲**：周期底部OCF可能异常低
3. **需结合商业模式**：特许经营型企业的资本开支含义与重资产企业不同
4. **短期波动大**：某些年份一次性大额资本开支会导致数据跳变，建议看3年平均
