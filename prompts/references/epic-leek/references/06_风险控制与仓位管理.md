# 风险控制与仓位管理

## 系统性风险识别

UP主关注的系统性风险信号：

| 信号 | 具体表现 | 应对 |
|------|----------|------|
| 指数月线破位 | 大盘月K跌破关键均线 | 降低总仓位，不抄底 |
| 行业轮跌 | 各行业轮番补跌，无避风港 | 只持有双负溢价资产 |
| 流动性收紧 | 央行政策转向、美元走强 | 保留现金，等待 |
| 地缘事件 | 非经济因素冲击 | 非必需不动 |

**核心原则：系统性风险不抄底。** 等月线/周线企稳后再考虑。

## 弹性差理论（Elasticity Difference）

UP主原创的仓位管理理论：

**概念**：不同资产在同一冲击下的波动幅度不同。仓位配置应考虑这种"弹性差异"。

### 仓位上限矩阵

| 商业模式等级 | 单只上限 | 逻辑 |
|-------------|---------|------|
| S级（国有垄断） | 20-25% | 确定性最高，可重仓 |
| A级（民营特许） | 15-20% | 确定性较高 |
| B级（成本/规模优势） | 10-15% | 有一定不确定性 |
| C级（无壁垒） | 5-10% | 不确定性最大，轻仓试探 |

### 弹性差的应用
- 高弹性（波动大）的资产：降低仓位
- 低弹性（波动小）的资产：可适当提高仓位
- **组合整体的弹性应保持在可接受范围**

## Walter Schloss 式分散

UP主推崇 Schloss 的分散投资理念：

- 持有 **15-30只** 股票
- 每只占比不超过总仓位的 **10-15%**
- 优势：单只踩雷不致命
- 前提：每只都经过完整的基本面验证，不是随机分散

> "Schloss 买一堆便宜货，每个都不重仓。这不是懒，这是承认自己可能判断错误。你买30只，错5只也无所谓。你集中买3只，错1只就是33%的损失。"

## 业务单元独立性风险

仓位决策中必须考虑"伪独立"风险：

### 风险检查清单
1. 子公司是否依赖母公司的品牌授权？
2. 子公司的客户是否主要来自母公司的业务生态？
3. 子公司的关键原材料是否由母公司集团内部供应？
4. 如果母公司出问题，子公司能否独立存活？

**如果答案是"不能独立"，该子公司的估值需要打折，仓位需要降低。**

## 股权结构风险

在事件驱动策略（特别是烟蒂股）中，股权结构决定了事件成功的概率：

| 股权特征 | 风险等级 | 说明 |
|----------|---------|------|
| 大股东持股 > 70% | 低风险 | 大股东有强动机和能力推动私有化 |
| 大股东持股 50-70% | 中风险 | 可能私有化，但成本较高 |
| 大股东持股 < 50% | 高风险 | 私有化难度大，可能需要联合要约 |
| 股权极度分散 | 最高风险 | 几乎不可能私有化 |

## 亏损处理原则

1. **逻辑正确+价格下跌 = 加仓机会**（前提：穿透回报率因下跌而提升）
2. **逻辑被证伪 = 止损离场**（不管亏多少）
3. **不因浮亏焦虑**：如果买入逻辑未变，浮亏只是暂时的价格偏差
4. **永远保留现金**：任何时候至少保留10-20%现金，用于极端机会

> "亏了不可怕，可怕的是你不知道为什么亏。如果你的逻辑对了但价格跌了，那是市场给你打折。如果你的逻辑错了，赶紧走。"
