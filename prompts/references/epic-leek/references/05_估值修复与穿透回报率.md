# 估值修复与穿透回报率

## "估值修复"不是"价值投资"

UP主明确区分自己的方法论与传统价值投资：

| | 传统价值投资 | UP主的估值修复 |
|---|---|---|
| 核心逻辑 | 买入好公司长期持有 | 判断当前定价是否错误，赚纠错的钱 |
| 持有期 | 越长越好 | 修复完成即可离场 |
| 关注重点 | 企业内在价值增长 | 市场预期与现实的偏差 |
| 卖出信号 | 高估时才卖 | 估值回到合理区间即卖 |

> "我不是价值投资者。我是做估值修复的。市场定价错了，我赌它会修正。修正完了，我走人。"

## 买入纪律：三重验证

进入一个仓位前必须同时满足：

### 1. 双负溢价确认
- 行业PE低于20年均值（行业负溢价）
- 个股PE低于行业当前PE（个股负溢价）

### 2. 基本面反转信号
- 经营现金流改善（最核心指标）
- 或：利空因素为非现金项目（商誉减值、资产重估等）
- 或：成本周期即将反转（原材料见顶）
- 或：行业供给侧出清（竞争者退出）

### 3. 穿透回报率达标
- 穿透回报率 ≥ 2倍无风险利率（最低门槛）
- UP主个人标准：≥ 4倍无风险利率

**三者缺一不可。** 便宜但基本面继续恶化 = 价值陷阱。基本面好但估值贵 = 不是估值修复的标的。

## 穿透回报率公式详解

```
穿透回报率 = (注销型回购 + 预期分红) / 成本市值
```

### 分子的计算
- **注销型回购**：只计已明确注销的回购，库存股不算
- **预期分红**：基于可支配现金流 × 预期派息率，不是基于会计利润
- 如果公司有明确的分红政策（如派息率≥50%），用政策指引
- 如果没有，用过去3年实际派息率的平均值

### 分母的选择
- **成本市值**：你的实际买入成本 × 总股本（评估个人持仓回报率）
- **当前市值**：当前股价 × 总股本（评估新买入是否值得）
- UP主一般用当前市值做初步筛选，用成本市值做持仓管理

## 卖出纪律

UP主的卖出框架：

1. **估值修复完成**：双负溢价消失（行业或个股估值回到均值以上）
2. **穿透回报率降至门槛以下**：股价上涨导致穿透回报率 < 2倍Rf
3. **基本面逻辑破坏**：原来判断的反转逻辑被证伪（如OCF持续恶化）
4. **出现更好的替代标的**：穿透回报率更高的双负溢价标的

> "不要因为涨了就卖，要因为它不便宜了才卖。也不要因为跌了就卖，要因为逻辑错了才卖。"

## 技术分析的态度

UP主对技术分析持明确的批判立场：

- **完全不用技术分析做买卖决策**
- 认为技术分析是"自我实现的预言"，本身不包含信息量
- 唯一参考：成交量缩量可以作为"卖盘枯竭"的辅助佐证

> "画线我是不画的。你跟我说什么支撑位阻力位，那都是事后画的。K线图不告诉你任何关于这家公司的信息。"

## 穿透回报率 vs 股息率

| | 穿透回报率 | 股息率 |
|---|---|---|
| 分子 | 注销回购 + 预期分红 | 实际已派分红 |
| 分母 | 市值 | 股价 |
| 前瞻性 | 前瞻（基于可支配现金流） | 后视（基于已发生分红） |
| 信息含量 | 高（考虑了回购+分红能力） | 低（可能不可持续） |

穿透回报率比股息率更有意义，因为它考虑了公司**能分多少**而非**选择分多少**。
