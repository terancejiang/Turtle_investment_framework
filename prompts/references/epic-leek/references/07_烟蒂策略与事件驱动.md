# 烟蒂策略与事件驱动

## 烟蒂投资的双因子系统

UP主将烟蒂投资简化为两个核心因子：

### 因子一：资产安全垫
- **PB < 0.3**（净资产打三折以上）
- 重点看**净有形资产**，扣除商誉和无形资产
- 净现金 > 市值 = 最强安全垫（相当于免费送你一家公司的经营业务）

### 因子二：正向经营现金流
- **OCF > 0**（公司仍在产生现金，不是在烧钱）
- 即使会计利润为负，只要OCF为正，公司就不会因为经营而破产
- OCF持续为负的烟蒂 = 真正的垃圾，远离

**双因子同时满足：** 资产打骨折 + 还在赚现金 = 典型的"被市场丢弃但还有价值的烟蒂"

## 事件驱动的三大类型

| 类型 | 触发条件 | 回报模式 | 风险 |
|------|----------|---------|------|
| **私有化** | 大股东提出全面要约 | 溢价30-50% | 要约可能被否决 |
| **要约收购** | 第三方收购或大股东增持 | 溢价20-40% | 价格竞争可能推高 |
| **清算/资产处置** | 公司决定出售资产或清盘 | 取决于资产价值 | 清算折价 |

## 股权结构分析（事件成功概率的关键）

在下注事件驱动前，必须分析大股东的意愿和能力：

### 私有化概率评估

```
大股东持股比例高 + PB极低 + 公司有净现金 = 私有化概率高
```

**逻辑链：**
1. 大股东持股高 → 需要收购的流通股少 → 私有化成本低
2. PB极低 → 私有化溢价在绝对金额上也不高
3. 公司有净现金 → 可以用公司自己的钱来私有化（杠杆收购）

### 反面情况
- 大股东持股低 → 需要大量资金收购 → 可能放弃
- 公司高负债 → 即使PB低也无法用自身现金私有化
- 监管障碍 → 国有企业私有化需要审批，不确定性更大

## 烟蒂投资的时间框架

| 阶段 | 时间 | 策略 |
|------|------|------|
| 建仓 | 双因子确认后逐步买入 | 不急，分批建仓 |
| 持有 | 6个月-3年 | 等待催化剂出现 |
| 退出 | 事件触发或估值修复 | 事件溢价兑现即走 |

> "烟蒂不是你买了明天就涨。你可能要等一年两年。但它的好处是，往下的空间很小（PB已经0.3了还能跌到哪去？），往上有事件驱动。"

## 推荐阅读

UP主推荐的烟蒂投资经典文献：

1. **《证券分析》** - Benjamin Graham（格雷厄姆）
   - 烟蒂投资的理论基础
   - 重点阅读：净流动资产价值（Net Current Asset Value）部分

2. **Walter Schloss 的投资记录**
   - 45年年化复合收益率约16%（同期标普500约10%）
   - 纯粹的烟蒂+分散策略

3. **Seth Klarman《安全边际》**
   - 系统化的安全边际理论
   - 强调"避免亏损比追求收益更重要"

## 烟蒂策略与主系统（龟龟投资法）的关系

| 维度 | 龟龟投资法（主系统） | 烟蒂策略（辅助系统） |
|------|---------------------|---------------------|
| 选股逻辑 | 双负溢价 + 基本面反转 | PB < 0.3 + OCF > 0 |
| 催化剂 | 估值修复 | 事件驱动（私有化等） |
| 持仓占比 | 70-80% | 20-30% |
| 分散度 | 5-15只 | 10-20只 |
| 单只上限 | 按商业模式分档 | 不超过5% |
