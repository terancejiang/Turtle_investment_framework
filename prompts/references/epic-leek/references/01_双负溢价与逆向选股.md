# 双负溢价与逆向选股体系

## 核心概念：负溢价分层

UP主将市场估值偏差分为两层：

| 层级 | 定义 | 判定方法 |
|------|------|----------|
| **行业负溢价** | 行业整体PE低于20年均值 | 当前行业PE / 行业20年平均PE < 1 |
| **个股负溢价** | 个股PE低于所在行业当前PE | 个股PE / 行业当前PE < 1 |

**双负溢价 = 行业负溢价 + 个股负溢价**，即行业整体被低估 + 该股票在行业内还被进一步低估。

## 四象限分类

```
             行业正溢价              行业负溢价
           ┌─────────────────┬─────────────────┐
个股正溢价  │ ④ 双正溢价       │ ② 行业负+个股正  │
           │ 最危险，远离      │ 需判断个股是否     │
           │                 │ 有独立逻辑        │
           ├─────────────────┼─────────────────┤
个股负溢价  │ ③ 行业正+个股负  │ ① 双负溢价       │
           │ 可能是价值陷阱    │ 最安全的买入区域   │
           │ 需深入基本面      │ 熊市生存能力最强   │
           └─────────────────┴─────────────────┘
```

## 逆板块错配（Davis Double Play 的触发条件）

这是UP主认为最罕见、最具爆发力的机会类型：

**三重条件同时满足：**
1. **行业负溢价**：市场整体看衰该行业
2. **个股负溢价**：该股票在行业内进一步被抛弃
3. **基本面与市场预期相反**：市场认为是利空，但实质是利好（或利空已被非现金减值消化）

**案例：蒙牛乳业（2024年报）**
- 行业PE处于20年低位 → 行业负溢价 ✓
- 蒙牛PE低于伊利等同行 → 个股负溢价 ✓
- 市场因大额减值恐慌卖出，但减值为非现金一次性项目，经营现金流反而改善 → 基本面反转 ✓
- 结论：逆板块错配 → 潜在Davis Double Play

## 系统性风险下的选股逻辑

当判断出现系统性下跌时（非个股问题）：

1. **非必需不动**：不因恐慌卖出基本面良好的持仓
2. **如需调仓**：向双负溢价资产集中（因为已经充分反映悲观预期，继续下跌空间有限）
3. **绝对回避**：双正溢价资产在系统性风险中跌幅最大

## 阈值指标参考

| 指标 | 阈值 | 说明 |
|------|------|------|
| 行业PE / 20年均值 | < 0.7 | 强行业负溢价信号 |
| 个股PE / 行业PE | < 0.8 | 强个股负溢价信号 |
| 双负持续时间 | > 6个月 | 充分消化悲观预期 |
| 换手率 | 显著缩量 | 卖盘枯竭信号 |

## 定性交易 vs 导数交易

UP主只做**定性交易**（qualitative trade）：
- **定性交易**：市场认为是负面的，你判断其实是正面的 → 赌方向反转
- **导数交易**：市场已认可是正面的，你赌增速会加速 → 赌斜率变化

> "我只做定性交易。市场说它差，我验证完发现它不差，这就够了。至于它好到什么程度，不是我关心的。"

定性交易的优势：安全边际天然存在（市场已经price in了负面预期）。导数交易风险更高，因为一旦增速不及预期，估值和盈利会双杀。
