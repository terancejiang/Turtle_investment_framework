# 实战案例库

## 案例1：中国食品（00506.HK）— 特许经营 + 少数股东调整

### 背景
- 可口可乐在中国的装瓶和分销商
- 中粮集团控股约65%

### UP主分析要点
1. **特许经营属性**：可口可乐品牌 = S级护城河
2. **少数股东陷阱**：报表OCF看起来很健康，但必须 ×0.65 才是归属母公司的
3. **净现金陷阱**：账面55亿现金，但流动比率 ≈ 1.0 → 现金参与经营滚动，不是安全垫
4. **定价权**：需求驱动型（可口可乐品牌溢价），但受限于原材料（糖、PET瓶）成本波动

### 关键教训
> 看到"净现金"不要兴奋，先看流动比率。看到"经营现金流强劲"不要兴奋，先算少数股东调整后的真实数字。

## 案例2：保利物业（06049.HK）— 双负溢价 + 净现金安全垫

### 背景
- 物业管理行业龙头之一
- 2024年物管行业整体估值处于历史低位

### UP主分析要点
1. **行业负溢价**：物管行业PE远低于20年均值（受地产暴雷连累）
2. **个股负溢价**：保利物业PE在行业内偏低
3. **净现金是真安全垫**：流动比率 ≈ 3.0，净现金约140亿 → 远超经营需要
4. **经营性市值极低**：总市值 - 140亿净现金 = 经营业务几乎白送
5. **穿透回报率（EV vs 市值）**：用市值算GG%更贴近实际回报

### 关键教训
> 物管行业被地产拖累是"逆板块错配"的典型场景。物管本身是轻资产、强现金流、高确定性的商业模式，不应该和重资产的地产开发商同折扣。

## 案例3：蒙牛乳业（02319.HK）— 逆向定性 + 非现金减值

### 背景
- 2024年报大额减值（现代牧业商誉等）
- 市场恐慌性抛售

### UP主分析要点
1. **减值是非现金项目**：不影响经营现金流
2. **OCF反而改善**：经营层面在变好，减值只是会计处理
3. **双负溢价**：乳业行业PE处于低位 + 蒙牛PE低于伊利
4. **逆板块错配**：市场认为减值 = 利空，但实质是一次性出清历史包袱
5. **典型的定性交易**：市场说它差（因为减值），我判断它不差（因为OCF改善）

### 穿透回报率计算（参见前期分析）
- 可支配现金流：经营现金流 - 全部Capex - 租赁偿还
- 分母用市值（非EV），因为蒙牛有净负债
- 回购+分红回报率 vs 无风险利率的倍数

### 关键教训
> 年报日暴跌往往是最好的买点，前提是你能区分"非现金减值"和"经营性恶化"。

## 案例4：华润啤酒（00291.HK）— 商誉减值恐慌

### 背景
- 大额商誉减值导致报表利润大幅下滑

### UP主分析要点
1. **商誉减值 = 非现金**：一次性会计处理
2. **经营现金流健康**：啤酒业务本身现金流稳定
3. **市场过度反应**：把非现金减值当作经营恶化来对待
4. **估值修复机会**：恐慌过后，市场会重新关注经营基本面

### 关键教训
> 商誉减值是价值投资者的朋友。它帮你清洗掉了一批不做基本面研究的持有人，给你一个更好的买入价。

## 案例5：福寿园（01448.HK）— 永久成本锁定

### 背景
- 中国最大的殡葬服务企业
- 核心资产：已购墓地土地

### UP主分析要点
1. **成本永久锁定**：土地一次性买入，成本不再变化
2. **售价持续提升**：殡葬服务的特殊文化属性支撑提价能力
3. **毛利率数学确定性**：成本固定 + 售价上涨 = 毛利率只能扩张
4. **商业模式 S 级**：刚需 + 文化壁垒 + 低竞争（政策管制）

### 关键教训
> 寻找"成本锁定型"企业。当你能数学推导出毛利率只能往上走时，这是最强的基本面确定性之一。

## 案例6：同仁堂国药（03613.HK）— 周期性成本反转

### 背景
- 中药材价格处于周期高位
- 毛利率被原材料成本压缩

### UP主分析要点
1. **周期性成本锁定**：中药材有种植和库存周期
2. **当前处于成本高位**：毛利率被压缩到低点
3. **品牌定价权不变**：同仁堂的售价不会因药材降价而降
4. **反转逻辑**：药材价格回落 → 毛利率恢复 → 利润弹性巨大
5. **国药品牌 = A级护城河**：百年品牌，客户信任度极高

### 关键教训
> 在成本周期高位买入品牌消费品，等成本回落赚利润弹性。前提：品牌方没有跟随降价的历史习惯。

## 案例7：中航信（TravelSky, 00696.HK）— 国有垄断典范

### 背景
- 中国民航信息系统唯一供应商
- 国有控股

### UP主分析要点
1. **绝对垄断**：政策壁垒，不可能有竞争者进入
2. **国有背景加持**：管理层即使不优秀，垄断利润也不会消失
3. **商业模式 S 级**：每张机票都要经过中航信系统
4. **弱周期性**：受航空旅客量影响，但长期趋势向上
5. **安全边际要求可降低**：因为确定性极高

### 关键教训
> 国有垄断是最不需要担心"护城河被攻破"的商业模式。分析重点从"能不能赚钱"变成"赚的钱怎么分配给股东"。

## 案例使用指南

在分析新标的时，可按以下方式参考案例库：

1. **识别最相似的案例** → 找到对应的分析框架
2. **检查关键差异** → 哪些方面与案例不同？
3. **应用关键教训** → 避免重复已知的分析陷阱
4. **更新案例库** → 新的分析完成后，将关键发现补充到案例库
